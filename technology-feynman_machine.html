Feynman Machine

The Feynman Machine is a machine intelligence model derived from intelligence principles of the mammalian neocortex, Deep Learning, Hierarchical Temporal Memory, and the Applied Mathematics of interacting Dynamical Systems. This technology is under development by Ogma Intelligent Systems Corp. [1]

Introduction

Many real-world problems cannot be tractably solved analytically or are extremely difficult to model precisely and thus fall under Dynamical Systems and Chaos Theory. Often solvers of these problems do not have access to the underlying mathematics, logic, internal dynamics (the differential or difference equations governing the update rules), control parameters which vary the global properties of the system, or the exact values of the observables of the system. Therefore, a simpler and in many ways more powerful method comes from Takens Theorum: reconstruction of the dynamics using signals from the observed system. The essence of the strategy is that sufficient information about the observed system is encoded in its signal to allow its reconstruction in an observing system using very simple computations. [3]

The mammalian neocortex is a Universal Modeller of Dynamical Systems. It is a Complex System, a hierarchy-like network of interconnected and coupled regions, each of which is operating as a dynamical system modelling a dynamical system. Cortical structures use a time-series of spatiotemporal sensorimotor inputs to:

  • generate a dynamical analogue of the system generating the inputs
  • forecast its short-term future
  • identify stable and slowly-changing characteristics of the system which indicate its hidden controlling parameters or state
  • model the nonstationarity of the system state
  • interact with the system using time series of motor outputs

Applied Mathematics

Dynamical Systems

A Dynamical System is a mathematical model whose dynamics are characterised by express update rules, typically differential equations in continuous systems, or difference equations in discrete time (Strogatz [2014]) [2].

A classical Dynamical System can be seen as a vector field or flow in its state (or phase) space, where the value of the vector field at each point is effectively a velocity vector indicating how the state at the point will change, tracing out a trajectory over time. In dynamical systems, causally interacting variables (say x, y and z) share a trajectory in the state/phase space (a geometrical manifold) [7]. At any given time, a dynamical system has a state given by a set of real numbers(a vector) that can be represented by a point in an appropriate state space [4]. An attractor is a set of numerical values toward which a system tends to evolve, for a wide variety of starting conditions of the system.

In 1963, Edward Lorenz developed a simplified mathematical model for atmospheric convection using 3 coupled differential equations. It is notable for having chaotic solutions for certain parameter values and initial conditions. [6]

  • x, y, z make up the system states and are abstractions of the state of the convecting fluid
  • x is proportional to the intensity of the convective motion
  • y is proportional to the temperature difference of ascending and descending currents
  • z is proportional to the departure of the vertical temperature profile from linearity
  • t is time
  • σ is a system control parameter. Lorenz used σ = 10
  • ρ is a system control parameter. Lorenz used ρ = 28
  • β is a system control parameter. Lorenz used β = 8/3

The evolution rule of the dynamical system is a function that describes what future states follow from the current state. Often the function is deterministic; in other words, for a given time interval only one future state follows from the current state; however, some systems are stochastic, in that random events also affect the evolution of the state variables. [4]

Chaos Theory

Chaos theory is the field of study in mathematics that studies the behavior of dynamical systems that are highly sensitive to initial conditions which yield widely diverging outcomes for the system [5]. The behavior of chaotic systems appears random even though these systems are deterministic, meaning that their future dynamics are fully defined by their initial conditions.

Takens Embedding Theorum

Floris Takens [1981], basing his study on results going back to Whitney [1936], proved that a system could be reconstructed (or embedded) in all important details using only a time-series of a single measurement from the system, or in other words by simply observing measurements of the real-world systems over time [3]. Takens' theorum is also refered to as a delay embedding theorum.

Φ = ( x(t), x(t-τ), x(t-2τ), ... x(t-kτ) ) of k > 2m + 1 observed values x

  • Φ is the vector of time-delayed measurements
  • x is the observed value at a certain time
  • t is time
  • τ is time delay
  • m is manifold dimension
  • k is the embedding dimension or number of measured points

The embedding model reconstructed from the time series observation is practically the same thing as the system being observed. Although the reconstruction is not visually identical to the original system, it is topologically identical, or diffeomorphic.

This mathematical property allows an agent to characterise an observed system and perform short-range forecasting of its future behaviour, without having access to the underlying causal mechanics or latent parameters of the system. [3]

A diffeomorphism is an isomorphism of smooth manifolds. It is an invertible function that maps one differentiable manifold to another such that both the function and its inverse are smooth. [diffeomorphism wiki]

Feynman Machine Architecture

A Feynman Machine is a collection of Dynamical Systems modules called regions, connected together in a ladder-like hierarchy of paired encoders and decoders. A Feynman Machine is connected to its external world via sensorimotor channels, each of which carries a (usually high-dimensional) time series signal of some kind. Each region is capable of adaptively learning, representing, predicting and interacting with the spatiotemporal, sensorimotor structure of its "world". [2]

Information flows both up the hierarchy (feature extraction), and down the hierarchy (prediction). Each encoder-decoder pair or predictor attempts to predict its inputs, ie the state of the layer below it, one timestep in advance. Lower layers use feedback information from the layers above to improve their own predictions. [2]

Each encoder-decoder pair is a kind of spatiotemporal autoencoder, receiving information, mapping to a hidden representation, and using that to predict the next frame of information. Predictions are compared with the actual next input, and this prediction error can be used to drive learning. [2]

Because of the generality of the theory of networked Dynamical Systems, we can choose to use many designs for the encoders and decoders, based on their different properties. The design we've found most useful in experiments and application is an online, spatiotemporal modification of the k-sparse autoencoder (Makhzani and Frey, 2013). [2]
